## 1. 아키텍처 및 핵심 모듈 개요

현재 옷장 및 쇼핑 어드바이저 시스템은 100% 로컬 오프라인 파이썬 환경에서 동작하며, 다음과 같은 계층 구조를 갖습니다:

```
[UI Layer: Streamlit (app.py)]
      │
      ├── (사용자 입력 프로필, TPO, 날씨, 추천 모드 전달)
      ▼
[Engine Layer: core/recommender.py (recommend_outfit, scan_missing_essentials)]
      │
      ├── (DB 데이터 로드 및 룰북 스코어링 매칭 연산)
      ▼
[Data Layer: JSON DBs (reference_styles, my_closet, shopping_candidates, outfit_history)]
```

## 2. 추가된 4대 핵심 기능 상세 사양

### ① 옷 태그 직접 수정 기능 (데이터 편집)
- **목적:** 자동 이미지 분류/색상 인식이 실패할 경우 사용자가 데이터를 보정하여 최적의 추천 정확도를 보장합니다.
- **구현 방식:**
  - 내 옷장 (`my_closet.json`): 카테고리, 성별, 색상, 계절, 스타일 직접 편집 지원.
  - 구매 후보 (`shopping_candidates.json`): 상기 태그에 더하여 **브랜드/쇼핑몰명(brand)**, **가격(price)**, **상품 링크 URL(url)** 추가 기재 지원.

### ② 구매 후보 살만한지 점수 (Buying Recommendation Metrics)
- **분석 지표:**
  - **어울리는 소장 의상 개수:** 내 진짜 옷장에서 해당 후보 색상과 어울리는 배색을 가진 의상 수.
  - **중복 의상 개수:** 동일 카테고리 및 색상을 가진 기소유 의상 수.
- **판단 알고리즘:**
  - `보류 (중복 우려)`: 중복 의상 개수 > 0 일 때.
  - `구매 추천 (높음) 🔥`: 중복 개수가 0개이고, 어울리는 소장 옷이 3개 이상일 때.
  - `구매 추천 (보통)`: 중복 개수가 0개이고, 어울리는 소장 옷이 1~2개일 때.

### ③ 부족한 옷 추천 (Missing Essentials Scanner)
- **분석 지표:** `my_closet.json` 인벤토리를 스캔하여 필수 베이직 아이템 구비율 진단.
- **진단 항목:**
  - **아우터:** 1개 미만 시 필수 아우터 제안 (네이비 블레이저 등). 2개 미만 시 추천 아우터 제안.
  - **하의:** 2개 미만 시 베이직 하의 제안 (블랙 슬랙스, 청바지).
  - **상의:** 3개 미만 시 베이직 상의 제안 (화이트 셔츠 등).
  - **밝은 뉴트럴 톤:** 화이트/베이지/그레이 계열의 총 의상 수가 3개 미만 시 밝은 중립색 추천.

### ④ 코디 저장 / 히스토리 (Outfit Logger)
- **저장 DB:** `outfit_history.json`
- **구현 기능:**
  - 코디 매칭 성공 시 `💾 오늘 코디 저장하기` 버튼 노출.
  - 클릭 시 날짜, TPO, 날씨, 매칭된 의류 상세 리스트(브랜드, 가격, 이미지 경로), 스타일명 저장.
  - 추천 화면 하단에서 역순(최신순)으로 이전 저장 코디를 선택하여 다시 보기 지원.

## 3. 기온 및 상세 추천 조건 고도화 (2026-05-20 추가)

### ⑤ 지능형 6단계 기온 연동 알고리즘
- **목적:** 날씨 정보에서 기온 숫자만 파싱하여 한파 및 폭염 상황에 완벽히 어울리는 의상을 정교하게 추천합니다.
- **구현 기온 구간:**
  - `freezing_winter` (4도 이하): 헤비 아우터 가점 (+30), 가벼운 아우터 감점 (-15).
  - `early_winter` (5~9도): 코트/패딩 가점 (+25), 여름 의류 감점 (-30).
  - `mild_season` (10~16도): 자켓/가디건 가점 (+25), 패딩 감점 (-20).
  - `warm_season` (17~22도): 얇은 가디건/자켓 가점 (+20), 패딩/코트 차단 (-30).
  - `early_summer` (23~27도): 반팔/시원한 소재 가점 (+20), 아우터 감점 (-25).
  - `hot_summer` (28도 이상): 반팔/민소매/반바지 가점 (+30), 긴팔/아우터 차단 (-30).

### ⑥ 상세 추천 조건 지능형 하이브리드 매핑
- **목적:** 사용자가 입력한 분위기(예: 포멀/클래식) 필터를 실제 스타일북 DB 라벨(`놈코어`, `캐주얼`, `스포티`, `로맨틱` 등)과 1:1로 매끄럽게 호환되도록 지능형 키워드 확장을 거쳐 가중치 및 미충족 진단을 계산합니다.
